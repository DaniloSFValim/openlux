## Como entrar nesta lista

1. Abra uma issue `[cidade] <nome>` no repositório.
2. Implante (tenant de consórcio ou instância soberana).
3. Envie um PR adicionando a linha acima, com link público do mapa.

> **Compromissos:** mapa público acessível, crédito ao projeto
> (DOI [10.5281/zenodo.21305310](https://doi.org/10.5281/zenodo.21305310)) e
> dados sob titularidade do próprio município.
